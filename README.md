# MOMS-PI_WMTS_2025
Code used to process and analyze all metatranscriptomic, 16s rRNA vaginal microbiome profile, and vaginal swab cytokine data from the MOMS-PI project.

**Requirements**
Fastp(v0.23.4)
Kraken2(v2.1.3 with k2_pluspf (2023-10-09) and additional in-house reference vaginal microbe genomes)
MultiQC(v1.17)
Salmon(v1.10.1)



R Studio(v4.4.0 or higher)
